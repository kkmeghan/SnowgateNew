# SnowgateNew

This repository contains the Snowgate website source files.

Files include the main `index.html`, assets in `css/`, `js/`, and `images/`.

To push locally to GitHub:

```powershell
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/kkmeghan/SnowgateNew.git
git branch -M master
git push -u origin master
```
